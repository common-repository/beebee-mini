== Description ==

` 提示：2.x.x版本之后的比比后台插件请从[比比网](https://beebee.work/2022/09/08/bi-bi-xiao-cheng-xu-hou-tai-cha-jian-beebeemini/)进行免费下载。 `


[比比小程序系列](https://beebee.work/yuan-ma/)-首创支持通过Wordpress后端所见即所得的方式DIY小程序。

如果您需要个性的小程序，比比的DIY功能完全符合您的胃口；
如果您对比比用户DIY的小程序喜欢，比比的主题导出导入功能也能方便您快速输出一样的小程序。

目前，比比小程序全系支持文章发布管理体系、积分体系、表单体系、会员交互体系、广告体系和DIY积木体系。
在此基础上：
比比爱书者支持图书管理体系、资源下载体系和图文连载体系；
比比图文社支持素材管理体系、音视频资源体系和资源下载体系；
比比小锦囊支持头像工具DIY、红包封面管理；
比比小睡眠支持音频列表管理；

以上，比比特别注重运营生态，已经打通公众号、小程序、视频号、小商店、直播、企业微信、客服系统。
特别适合大学生作为毕业作品、设计师展示作品以及自媒体用户使用。

@小程序目前支持微信端，未来将依次支持字节端（抖音等）、QQ端、百度端、支付宝端

== Installation ==

1. 上传 `beebee-mini`目录 到 `/wp-content/plugins/` 目录
2. 在后台插件菜单激活该插件
3. 进入该插件主页启用 `爱书者`等您喜欢的主题
4. 主题启用后如需配套的插件支持会进行提示，按照提示安装相关插件即可
5. 进入该插件主题设置进行相关参数设置，包括菜单、布局、广告等参数设置
6. 添加专题、书目、语录等主题相关文章内容
7. 详细的配置教程前往比比帮助文档[https://doc.beebee.work/](https://doc.beebee.work/)
